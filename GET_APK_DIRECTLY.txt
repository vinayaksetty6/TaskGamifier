                       WHAT HAPPENS BEHIND THE SCENES
================================================================================

When you upload to GitHub:

1. GitHub's servers detect your code
2. GitHub starts automatic build (GitHub Actions)
3. Build server downloads JDK (Java Development Kit)
4. Build server downloads Android SDK
5. Build server downloads all dependencies
6. Kotlin code is compiled to Android bytecode
7. APK file is packaged
8. APK is made available for download
9. You download it
10. You copy to phone
11. Phone installs it
